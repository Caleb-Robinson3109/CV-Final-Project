# /cs5404

## This project aims to add further refinement of KITRO's predicitions using a random forest, while providiing the user an easy way to visualize the model

## Requirements
* Python 3.6
* PyTorch 1.10.1

## Stetup
```bash
python3.6 -m venv venv
source venv/bin/activate
pip install -r cs5404/requirments.txt
```
Go to [KITRO README](../KITRO_README.MD) and download the datasets. Place the data in /cs5404/data and /data. NOTE we do not control the Google Drive where the datasets are stored, they may become unaviable at any point.
